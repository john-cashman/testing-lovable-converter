Transactions

The page Transactions, allows you to generate a report of all the payment and refund transactions made using Enviso Pay. The user should have permission to 'View transactions' (Admin app).

Note

A license is required to use Enviso Pay for processing payments. See further, Licenses

Select a date and click Export transactions.

Generating the report might take some time. It is recommended that you refresh your page. Once generated, you will receive a notification that your transaction log is ready to download. Click on the notification to download it.
